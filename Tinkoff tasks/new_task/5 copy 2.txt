Напиши программу, которая решает следующую задачу:
На вход подаётся список из чисел, нужно определить количество нормальных отрезков для данного списка. Нормальным отрезком считается если суммарное изменение баланса с i по j день равно нулю.


Ввод
3
42 -42 42

Вывод
3

Вввод
4
1 2 3 -6

Вывод
1

Ввод
5
-1 1 2 -3 6

Вывод
6


Напиши программу на пайтоне, которая решает следующую задачу:
На вход подаётся список из чисел, нужно определить количество нормальных отрезков для данного списка. Отрезок из дней [l, r] считается нормальным, если внутри данного подотрезка можно найти разумный подотрезок. Отрезок из дней [i, j] считается разумным, если суммарное изменение баланса с i по j день равно нулю, т.е. a1 + ai+a1 + ... aj = 0.


Ввод
42 -42 42
Вывод
3

Вввод
1 2 3 -6
Вывод
1

Ввод
-1 1 2 -3 6
Вывод
6
