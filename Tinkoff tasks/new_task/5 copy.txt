Напиши программу на пайтон, которая бы решала следующую задачу:
Саша ведёт бюджет и анализирует, как изменялся баланс на его счету. Он выписал числа a1,a2, ... an - изменения за последние n дней.
Отрезок из дней [i, j]  Саша счиает разумным, если суммарное изменения баланса с i-ого по j-ый ендь равно нулю, т.е. ai + ai+1 + ... + aj = 0.
Отрезок из дней [l, r] считается нормальным, если внутри данного отрезка можно найти разумный подотрезок.

Помоигте Саше проанализировать эффективность ведения бюджета и посчитайте количество нормальных отрезков в массиве изменений баланса его счета.

Формат входных данных

Первая строка содержит число n(1 <= n <= 2 * 10^5) - количество дней, для которых Саша записывал изменения баланса.
Вторая строка содержит n чисед a1, a2, ... an (-10^9 <= ai <= 10^9) - значения изменений баланса.

Формат выходных данных

Выведите одно число - количество нормальных отрезков для данного массива.

Пример тестовых данных, для проверки программы:
Ввод
3
42 -42 42

Вывод
3

Вввод
4
1 2 3 -6

Вывод
1

Ввод
5
-1 1 2 -3 6

Вывод
6
